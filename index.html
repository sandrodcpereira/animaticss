Quicky preview and generate code for frame animations on the web.

This web tool is actively being worked on and won't necessarily work properly just yet!

1. Upload your own animation or enjoy the built-in ones.

To preview one of your own animations in Animaticss, you first need to combine all the frames into a horizontal strip.

 

2. Adjust the number of frames until it plays correctly.

Number of frames

3. Tweak the speed until it feels right.

4. Copy the code (don't forget to replace the image path) and add it to your CSS file!

.animation-container {
	width: [sourceWidth / noFrames];
	height: [sourceHeight];
	background-image: url('your-image.png');
	background-size: cover;
	animation: preview [animationSpeed] steps([noFrames]) infinite;
}

@keyframes preview {
	from {background-position-x: 0px;}
	to {background-position-x: [sourceWidth - [sourceWidth / noFrames]];}
}
